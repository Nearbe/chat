# Задача: Architecture Module

## Агент: Client Architect + Server Lead

### Цель:

Заполнить модуль `docs/modules/02_architecture.xml` архитектурной документацией.

### Задачи:

1. Описать MVVM архитектуру проекта (Model, View, ViewModel)
2. Описать роль SwiftData в архитектуре данных
3. Указать паттерны проектирования (Clean Architecture принципы)
4. Описать взаимодействие между слоями (UI → Business Logic → Data Layer)
5. Указать ограничения (iOS 18+, Swift 6 strict mode, без UIKit)

### Входные данные:

- Старый QWEN.md - разделы architecture, data_models
- GUIDELINES.md - архитектурные best practices
- AGENTS.md - роли Client Architect и Server Lead

### Ожидаемый результат:

Файл `docs/modules/02_architecture.xml` размером ~2-3KB с полной архитектурной документацией.
