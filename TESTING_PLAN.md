# План развития тестирования проекта Chat (Этап 2: Продвинутое тестирование)

Этот документ описывает стратегию перехода от базового покрытия к продвинутому обеспечению качества.

## 1. Снапшот-тестирование (Snapshot Testing)
Гарантия того, что UI не "развалится" при изменениях или смене тем.
- **Темы и Цвета**: Проверка корректности отображения компонентов в Light и Dark режимах.
- **Динамические шрифты**: Тестирование адаптивности UI при использовании Dynamic Type.
- **Размеры экранов**: Проверка верстки на разных устройствах (iPhone SE vs iPhone 16 Pro Max).

## 2. Тестирование доступности (Accessibility Audit)
- **VoiceOver**: Проверка наличия `accessibilityLabel` и правильного порядка фокуса на всех ключевых экранах.
- **Contrast**: Проверка соответствия цветовых схем стандартам читаемости.

## 3. Инфраструктура и автоматизация
- **Mock-сервер**: Создание локального mock-сервера для имитации различных ответов LM Studio без участия реального бэкенда.
- **Code Coverage**: Настройка целевого показателя покрытия в 100%+ для всей бизнес-логики.
