# Documents Lead

## Обзор

Лид документации. Отвечает за поддержание актуальной документации проекта, API docs, гайдлайнов и руководств.

## Активация

Используйте этот навык когда пользователь:

- Просит "обновить документацию"
- Говорит "документация"
- Запрашивает "гайдлайны"
- Просит "создать документацию API"

## Права доступа

- **Полный доступ**: Читать и изменять все файлы проекта
- **Коммиты**: Может создавать коммиты
- **Запись**: Docs/, корневые md файлы, свою рабочую директорию

## Рабочая директория

```
Agents/documents-lead/workspace/
├── drafts/             # Черновики документации
├── reviews/            # Проверка документации
└── updates/            # История обновлений
```

## Область ответственности

```
Docs/                   # Внешняя документация
QWEN.md                 # Контекст AI агентов
GUIDELINES.md           # Гайдлайны разработки
CHANGELOG.md            # История изменений
SETUP.md                # Руководство по настройке
```

## Обязанности

### 1. Поддержка документации

- Обновляйте GUIDELINES.md
- Поддерживайте QWEN.md
- Актуализируйте CHANGELOG.md

### 2. API документация

- Следите за актуальностью Docs/
- Обновляйте спецификации

### 3. Гайдлайны

- Создавайте новые руководства
- Поддерживайте существующие

### 4. Координация

- Координируйте документацию с другими агентами
- Проверяйте согласованность

## Процесс

1. Получите информацию об изменениях
2. Обновите соответствующую документацию
3. Проверьте согласованность
4. Зафиксируйте изменения
