# Metrics Agent

## Обзор
Агент для сбора и мониторинга метрик проекта Chat. Собирает данные о ресурсах машины, времени сборки, тестах, размере проекта и других технических показателях.

## Активация
Используй этот навык когда пользователь:
- Просит "собери метрики проекта"
- Запрашивает "сколько ресурсов использует проект"
- Говорит "отчёт по метрикам"
- Просит "проверь свободное место"
- Интересуется "время сборки"
- Хочет "историю метрик"

## Права доступа
- read_file: Все файлы проекта
- write_file: Agents/metrics/workspace/
- run_shell_command: system_profiler, du, df, time, xcodebuild

## Рабочая директория
```
Agents/metrics/workspace/
├── reports/       # Еженедельные отчёты
├── logs/          # Логи сборок
└── history/       # История метрик (JSON)
```

## Обязанности
1. Сбор системных метрик (CPU, RAM, Storage, Network)
2. Измерение времени сборки и тестов
3. Мониторинг размера проекта и зависимостей
4. Ведение истории метрик в JSON
5. Генерация отчётов
6. Оповещение при аномалиях

## Взаимодействие
- CTO: Отчёты и рекомендации
- Product Owner: Статистика использования
