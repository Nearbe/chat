# Текущее состояние продукта

**Дата:** 24 февраля 2026  
**Версия:** 1.0.0

---

## 1. Продукт

**Chat** — iOS-приложение для чата с локальными LLM (LM Studio / Ollama).

### Что умеет

| Функционал | Статус | Описание |
|------------|--------|----------|
| Чат с LLM | ✅ Готово | Отправка сообщений, получение ответов через SSE streaming |
| Выбор модели | ✅ Готово | Список доступных моделей, загрузка/выгрузка |
| История чатов | ✅ Готово | SwiftData персистентность, несколько сессий |
| Экспорт в Markdown | ✅ Готово | ShareSheet для экспорта |
| Безопасная авторизация | ✅ Готово | Keychain для токенов |
| Network мониторинг | ✅ Готово | Pulse для отладки сети |
| Device Authentication | ✅ Готово | 3D Shield View для ввода токена |

---

## 3. Архитектура

```
MVVM + SwiftData + Apple Keychain

Features/
├── Chat/       # Главный экран чата
├── History/    # История сессий
├── Settings/   # Выбор модели
└── Common/     # Общие компоненты

Services/
├── Chat/       # Бизнес-логика чата
├── Network/    # HTTP + SSE
└── Auth/       # Keychain + Device Auth
```

## 4. Сильные стороны

- **Зрелая инфраструктура:** 100% тестовое покрытие, SwiftLint, автоматический check/ship
- **Безопасность:** Токены в Keychain, авторизация устройств
- **Качество кода:** Swift 6.0, строгий линтинг, документация
- **Автоматизация:** One-click deployment на реальное устройство
