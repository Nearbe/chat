# План развития Chat App

> Vibe-coding студия для управления AI-агентами

## Видение

**Приложение для вайб-кодинга**, которое максимально упрощает управление и разработку проекта. Реальный разработчик
остаётся максимально мобильным.

## Экраны и Навигация

### Сетка экранов

```
[Metrics] ← [Chat] ← (текущий)
    ↑
  [Roles]
```

- **Заполнение**: справа → слева → сверху → снизу
- **Угловые** — отсутствуют
- **Навигация**: D-pad (↑↓←→) или свайпы от краёв

### Список экранов

| # | Экран   | Описание                               |
|---|---------|----------------------------------------|
| 1 | Chat    | Чат с LLM (уже реализовано, доработки) |
| 2 | Roles   | Доска ролей с задачами для агентов     |
| 3 | Metrics | Статистическая доска проекта           |

---

## Фичи

### 1. Spatial Navigation (Фундамент)

| #   | Задача           | Описание                                             |
|-----|------------------|------------------------------------------------------|
| 1.1 | D-pad Controller | Компонент с 4 кнопками (↑↓←→) для навигации          |
| 1.2 | Screen Grid      | Сетка экранов с movement constraints (без углов)     |
| 1.3 | Grid Navigator   | Логика перемещения по сетке, текущий экран подсвечен |
| 1.4 | Swipe Gestures   | Альтернатива d-pad — свайпы от краёв                 |

### 2. Roles Board

| #   | Задача                | Описание                                               |
|-----|-----------------------|--------------------------------------------------------|
| 2.1 | Roles Grid View       | Сетка карточек ролей                                   |
| 2.2 | Role Card             | Карточка роли (название, иконка, индикатор активности) |
| 2.3 | Task List (collapsed) | Список задач внутри роли (скрыт/раскрыт)               |
| 2.4 | Task Queue            | Очередь задач — каждая роль берёт по порядку           |
| 2.5 | Task Status           | Статусы: Pending → In Progress → Done                  |
| 2.6 | Task Assignment       | Постановка задачи в очередь                            |
| 2.7 | Roles Data Model      | Модель данных для ролей и задач (SwiftData)            |

### 3. Metrics Board

| #   | Задача                 | Описание                     |
|-----|------------------------|------------------------------|
| 3.1 | Metrics Dashboard View | Экран для отображения метрик |
| 3.2 | Metric Card            | Компонент карточки метрики   |
| 3.3 | Metrics Data Model     | Модель данных для метрик     |
| 3.4 | Metrics Service        | Сервис сбора/расчёта метрик  |

### 4. Chat (Доработки)

> Подлежит уточнению при детальном рассмотрении

---

## Порядок реализации

1. **Spatial Navigation** — фундамент для всех экранов
2. **Roles Board** — основная фича
3. **Metrics Board** — дополнительная функциональность
4. **Chat Improvements** — доработки по результатам review
